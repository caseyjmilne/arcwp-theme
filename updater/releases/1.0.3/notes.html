ARC\WP Theme

Version 1.0.3 | Released: October 2, 2025

What's New in 1.0.3

This release introduces package support with a beautiful grid archive layout, enhanced branding throughout the theme, and improved header/footer navigation components.

Changelog

New Features

  • Added complete package post type support with custom archive template
  • Implemented responsive package grid layout with featured images
  • Created package archive template with proper asset loading
  • Added support for package metadata and custom fields

Improved Enhancements

  • Updated site header with modern ARC\WP branding
  • Redesigned footer with ARC\WP logo and updated navigation structure
  • Enhanced footer links with proper WordPress URL routing
  • Updated footer tagline to "where speed meets precision"
  • Improved social media integration in footer (X/Twitter)
  • Refined footer copyright to display "ARC Software"

Fixed Bug Fixes

  • Corrected package image asset paths and loading
  • Fixed footer navigation menu item URLs
  • Resolved branding consistency across header and footer

Requirements

  • WordPress 6.0 or higher
  • PHP 7.4 or higher
  • MySQL 5.7 or higher

Installation Notes

Important: This update will be automatically downloaded and installed. After updating, please clear any caching plugins you may have installed to ensure all changes take effect.

The update process is automatic and safe. Your theme customizations and settings will be preserved.
