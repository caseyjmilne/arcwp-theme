ARC WP Theme

Version 1.0.1 | Released: October 1, 2025

What's New in 1.0.1

This release includes bug fixes, performance improvements, and enhanced functionality for your ARC WP powered sites.

Changelog

New Features

  • Added automatic theme update system via GitHub releases
  • Integrated Tailwind CSS output with proper cache busting
  • Enhanced menu system with better navigation support

Improved Enhancements

  • Optimized asset loading for better performance
  • Improved theme versioning for cache management
  • Better compatibility with WordPress 6.7+

Fixed Bug Fixes

  • Resolved stylesheet versioning issues
  • Fixed post thumbnail support initialization
  • Corrected update checker integration path

Requirements

  • WordPress 6.0 or higher
  • PHP 7.4 or higher
  • MySQL 5.7 or higher

Installation Notes

Important: This update will be automatically downloaded and installed. After updating, please clear any caching plugins you may have installed to ensure all changes take effect.

The update process is automatic and safe. Your theme customizations and settings will be preserved.
